Agent Names + Functionality + Dependencies

Core

A1 — ICP Strategist

Main functionality: Defines ICP policy: segments, title rules, keywords, exclusions, and threshold.

Input: Source lead count + product positioning context

Output: ICP policy artifact used by signal/scoring agents

Depends on: None (source agent)

Feeds: A2

Core

A2 — Signal Miner

Main functionality: Transforms raw leads into normalized signal features.

Input: Source leads + ICP policy + external source adapters (A7/A9)

Output: Per-lead structured signals (keywords, role, region, profile)

Depends on: A1, A7, A9

Feeds: A3

Core

A3 — Lead Scorer

Main functionality: Applies deterministic scoring and acceptance/rejection logic.

Input: Signal records from A2

Output: Accepted and rejected lead lists with reasons and scores

Depends on: A2

Feeds: A4, A11

Core

A4 — Personalization Writer

Main functionality: Generates personalized email drafts and blog-angle hooks.

Input: Accepted leads + enrichment (A11) + competitive narrative cues (A12)

Output: Outreach drafts and content angles

Depends on: A3, A11, A12

Feeds: A5

Core

A5 — QA Approver

Main functionality: Quality-gates drafts and retries low-quality content once.

Input: Drafts from A4

Output: Approved/rejected assets with QA notes

Depends on: A4

Feeds: A6

Core

A6 — GTM Orchestrator

Main functionality: Runs chain control, validation checks, and publishes run artifacts.

Input: All upstream core outputs

Output: Run summary, validation status, and artifact index

Depends on: A5

Feeds: A12

New

A7 — OSS Community Pipeline

Main functionality: Scans GitHub stargazers and ranks community prospects.

Input: GitHub stargazers + profile enrichment

Output: Community prospect list (tiered) feeding A2/A11

Depends on: None (source agent)

Feeds: A2, A11

New

A8 — Benchmark Signal Watchdog

Main functionality: Monitors benchmark chatter and recent benchmark papers.

Input: HN benchmark queries + arXiv feed

Output: Threat/event signals for positioning via A12

Depends on: A10

Feeds: A12

New

A9 — Reddit Intent Radar

Main functionality: Finds high-intent Reddit threads from problem/verification language.

Input: Reddit search results by GTM intent queries

Output: High-intent community signals feeding A2

Depends on: None (source agent)

Feeds: A2

New

A10 — HackerNews Trend Radar

Main functionality: Collects and deduplicates HN trend threads by strategic keywords.

Input: HN Algolia keyword searches

Output: Thread-level trend signals feeding A8/A12

Depends on: None (source agent)

Feeds: A8, A12

New

A11 — ICP Enrichment Resolver

Main functionality: Enriches accepted leads with persona/domain/link fields.

Input: Accepted leads (A3) + source CSV + A7 context

Output: Enriched profile records feeding A4

Depends on: A3, A7

Feeds: A4

New

A12 — Competitive Intel Synthesizer

Main functionality: Builds competitor threat and counter-positioning intelligence.

Input: Watchdog signals (A8/A10) + orchestration context (A6)

Output: Competitive narratives feeding A4 and strategy

Depends on: A8, A10, A6

Feeds: A4
